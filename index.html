PolarDB for PostgreSQL

A cloud-native database service independently developed by Alibaba Cloud

Getting Started Architecture Introduction

Flexible Scalability

You can use the service to scale out a compute cluster or a storage cluster based on your business requirements.

Millisecond-level Latency

Write-ahead logging (WAL) logs are stored in the shared storage. The LogIndex technology provided by PolarDB features two record replay modes: lazy replay and parallel replay.

HTAP

HTAP is implemented by using a shared-storage-based massively parallel processing (MPP) architecture.
